# Laravel Migration & Seeder

Questo progetto simula un tabellone ferroviario dinamico, focalizzandosi sulla gestione strutturata del database tramite le funzionalità avanzate di **Laravel**. L'applicazione gestisce un sistema di filtraggio dei treni in partenza, con dati generati interamente in modo programmatico.

* **Laravel Seeders**
* **Faker**
* **Eloquent ORM**
